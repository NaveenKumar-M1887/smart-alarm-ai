# 🌙 Smart Alarm AI

Smart Alarm AI is a web-based application that helps users wake up at the perfect time based on natural sleep cycles.

---

## 🚀 Features

- ⏰ Sleep cycle-based wake-up prediction
- 📊 Quality analysis (Average, Good, Excellent)
- 📈 Percentage-based sleep efficiency
- 🔔 Smart alarm with sound & interaction
- 🧠 Question-based alarm stop system
- 📜 History tracking with clear option

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Logic:** Sleep cycle algorithm (90-minute cycles)

---

## ⚙️ How It Works

1. User enters sleep time  
2. System calculates wake-up times using 90-minute cycles  
3. Assigns quality levels  
4. Displays results  
5. Alarm triggers at selected time
